# 学生成绩管理系统 python251

## 项目概述

学生成绩管理系统是一个基于Python的控制台应用程序，旨在实现对学生信息与成绩的有效管理。系统采用文件存储方式，提供信息的维护、查询、统计和排序等功能。

## 技术栈

- Python

## 功能模块

### 信息维护

- 增加学生信息
- 删除学生信息
- 修改学生信息

### 信息查询

- 按姓名查询
- 按学号查询

### 成绩统计

- 输入课程名和分数段，统计此分数段的学生情况

### 排序

- 指定课程名，按成绩升序或降序排列学生数据

## 系统角色

- 管理员：负责对学生信息的维护、查询和成绩统计分析

## 运行环境

- Python环境

## 部署步骤

1. 准备Python运行环境
2. 导入项目代码
3. 运行程序

## 核心亮点

- 纯控制台操作，简洁高效
- 功能模块化，易于扩展与维护
- 使用文件存储数据，无需数据库支持，易于部署
- 提供成绩统计与排序功能，方便分析学生成绩分布情况
